# XPUMP Configuration Tool

This tool is used to configure and communicate with our pump system.

## Requirements

- **OS:** Windows, tested on Windows 10/11
- **Hardware:** USB/Serial connection to the pump device

## Usage

1. Connect the pump to your computer via USB.
2. Start `XPConfig.exe`.
3. Select the correct COM port from the dropdown menu.
4. Configure parameters as needed.

## Notes

- **Drivers:** Make sure the correct drivers for your USB-to-Serial adapter are installed.
- **Port Conflict:** Close other programs, such as terminal emulators, that may use the same COM port.
- **Security:** If Windows SmartScreen shows a warning, click **More info** and **Run anyway**.
